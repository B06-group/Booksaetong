## 👀 프로젝트 계기
프로젝트 기획단계에서 주제에 대해 논의 중에 **중고 거래 플랫폼**에 대해 만들어 보자는 의견이 통일되어서 주제를 좁혀서 중고 책 거래 플래폼으로 채택했어요!

## 🛠️ 주요기능

### ✅ 중고 책 판매
중고로 팔고 싶은 책을 `제목`, `내용`, `거래 위치`, `금액`, `사진`을 입력해서 올리고, 댓글로 구매의사를 표현할 수 있습니다.
또한, 내가 작성한 게시글의 수정, 삭제가 가능하고 내가 작성한 댓글의 수정, 삭제가 가능합니다.

## 🏹 Trouble Shooting

### Kakao Map API 다루는 과정
> - 증상 : Places 오류
> - 해결 방법 : 전체 layout에 script로 Kakao 지도를 처음 렌더링 되게 하면 해결이 된다.

### 리스트조회 페이지 필터링 로직
> - 증상 : 여러테이블에서 조회하다보니 성능적으로 안좋고 코드량도 많아졌다.
> - 해결 방법 : 슈파베이스의 쿼리조인을 이용해서 한번에 조회할 수 있었다.
